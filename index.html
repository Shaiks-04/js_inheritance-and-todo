1.Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype


Code:

        const parent={
            fname:"shaik",
            method:function mtd(){
                console.log("inside parent object method "+this.fname);
            }
        }
        const child={
            __proto__ : parent
        }
        
        child.method();

Output:

inside parent object method shaik

2.Write code to explain prototype chaining.

Code:

        const parentObj = {
        name: 'Parent',
        sayHello: function() {
            console.log(`Hello, I'm ${this.name}.`);
        }
        };
        const childObj = Object.create(parentObj);
        childObj.name = 'Child';
        childObj.sayHello();

Output:

Hello, I'm Child.
